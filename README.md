# UNOW AI 使用指南

<aside>
💡 轻量级AI工具集，可保存常用prompt结合各种数据采集方案进行使用（暂时仅支持文本一些相关采集）
</aside>

---

**APP快捷键：**
ALT/OPTION+A：打开窗口
ALT/OPTION+S：OCR，截图捕获文字
ALT/OPTION+M：捕获选中文字打开窗口
